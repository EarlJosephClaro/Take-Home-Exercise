    How it works & API reference

    Every short link maps to one original URL. Opening a short link records a hit for the current day (UTC) and redirects you on. Stats show all-time hits plus a per-day breakdown for the last 30 days.

    Endpoints

    • POST /shorten body { "url": "https://…" } → 201 { short_code, short_url }
    • GET /:short_code 302 redirect to the original URL (records a hit)
    • GET /stats/:short_code 200 total hits + 30-day daily breakdown
    • GET /health 200 { status: "ok" }
